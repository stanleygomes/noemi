## [1.0.0] - 2024-02-14

### Added

- Initial release of Noemi Theme
- Dark theme with carefully crafted color palette
- Light theme for daytime coding
- Full support for Islands UI (modern JetBrains UI)
- Support for classic IDE themes
- Editor color schemes for both dark and light themes
- Comprehensive UI element styling
- Support for IntelliJ IDEA 2022.3+ (build 223+)
- Compatible with all JetBrains IDEs

### Features

- Beautiful dark color palette inspired by Catppuccin
- Clean light theme with excellent readability
- Syntax highlighting for multiple languages
- Custom colors for:
  - Buttons and actions
  - Editor components
  - Tool windows
  - Search and navigation
  - Trees and tables
  - Progress bars and scrollbars
